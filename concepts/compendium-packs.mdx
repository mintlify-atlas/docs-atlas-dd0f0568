Compendium packs are Foundry VTT's content libraries — read-only (in production) collections of documents that are distributed with a system or module. The PF2e system ships **94 compendium packs** stored under [`packs/pf2e/`](https://github.com/foundryvtt/pf2e/tree/master/packs/pf2e), covering every aspect of the Pathfinder 2e rules.

Each pack is a directory of YAML source files that get compiled into a LevelDB database at build time. You reference compendium entries in text using the inline link format:

```
@Compendium[pf2e.spells.Fireball]{Fireball}
```

## Character Building

These packs contain the content players need to build and advance characters.

| Pack | Contents |
|------|----------|
| `ancestries` | All published ancestries (Human, Elf, Dwarf, Gnome, …) |
| `ancestry-features` | Ancestry features and heritage feats granted at character creation |
| `heritages` | Heritage items for each ancestry (Half-Elf, Ancient Elf, Razortooth Goblin, …) |
| `backgrounds` | All published backgrounds |
| `classes` | Class documents (Fighter, Wizard, Rogue, …) |
| `class-features` | Class feature items granted at each class level |
| `feats` | All general feats, skill feats, class feats, and archetype feats |
| `spells` | Every published spell, cantrip, and focus spell |
| `deities` | Deities and philosophies with their divine font, domains, and cleric spells |
| `familiar-abilities` | Familiar and master abilities for the familiar subsystem |
| `boons-and-curses` | Divine boons and curses awarded by deities |
| `pathfinder-society-boons` | Organized Play boons for PFS players |

## Equipment

All physical items are in the `equipment` pack, which contains weapons, armor, adventuring gear, consumables, shields, and treasure.

| Pack | Contents |
|------|----------|
| `equipment` | The primary equipment pack — weapons, armor, shields, adventuring gear, consumables, alchemical items, magical items, and treasure |
| `equipment-effects` | Effects that activate when equipment is used (e.g., potion of healing effect) |
| `vehicles` | Vehicle stat blocks (ships, wagons, siege engines) |
| `iconics` | Paizo iconic pregenerated characters with full gear |
| `paizo-pregens` | Official Paizo pregenerated characters |

## Effects and Conditions

Effects are auto-applied items that represent ongoing mechanical states. Conditions are a special subset that follow PF2e's official condition rules.

| Pack | Contents |
|------|----------|
| `conditions` | All 47 official PF2e conditions (Frightened, Prone, Clumsy, Stunned, etc.) |
| `feat-effects` | Effects applied when feat abilities activate |
| `spell-effects` | Effects applied by spells (duration tracking, persistent bonuses) |
| `campaign-effects` | Effects used by campaign-specific subsystems (kingdom events, etc.) |
| `other-effects` | Miscellaneous effects not tied to a specific feature |

## Hazards and Actions

## Using the Compendium Browser

In-game, you can search all installed compendium packs using the **Compendium Browser** (the book icon in the scene toolbar, or via the sidebar). The browser provides filtered views for:

- **Spells** — filter by tradition, school, rank, trait, and action cost
- **Equipment** — filter by item type, level, price range, and traits
- **Feats** — filter by type, prerequisite, skill, and ancestry
- **Actions** — browse exploration, downtime, and encounter actions
- **Bestiary** — filter creatures by level, traits, alignment, and source
- **Hazards** — filter by level, complexity, and traits

<Tip>
  The Compendium Browser indexes all active compendium packs—including those from installed modules. Use **Game Settings → Compendium Browser Settings** to hide packs you don't want included in search results.
</Tip>

## Inline Compendium Links

When authoring journal entries, item descriptions, or NPC ability text, you can embed live links to compendium entries. Clicking the link in play opens the document sheet:

```
@Compendium[pf2e.spells.Fireball]{Fireball}
@Compendium[pf2e.conditions.Frightened]{Frightened 2}
@Compendium[pf2e.feats.Assurance]{Assurance (Athletics)}
```

The format is `@Compendium[<system-id>.<pack-name>.<Document Name>]{Display Text}`.

## Contributing Compendium Content

Compendium source files live in `packs/pf2e/<pack-name>/` as individual YAML files. When contributing or editing compendium content:

<Steps>
  <Step title="Edit the YAML source">
    Each document is a single `.yml` file named after the document's slug. Edit the YAML directly — do not edit the compiled database.
  </Step>
  <Step title="Follow naming conventions">
    File names must be the document's slug (lowercase, hyphenated). The document's `name` field should match the official Paizo publication exactly, including capitalization.
  </Step>
  <Step title="Use compendium UUIDs for cross-references">
    When one compendium entry references another, use the full `Compendium.pf2e.<pack>.<id>` UUID format. Avoid hard-coding item IDs that may differ between worlds.
  </Step>
  <Step title="Run the build">
    After editing YAML source, run `npm run build:packs` to compile the sources into the LevelDB databases that Foundry reads.
  </Step>
</Steps>

<Warning>
  Never edit the compiled pack databases in `packs/pf2e/<pack-name>/_source` directly in a production world. Always work from the YAML source files in the repository and rebuild.
</Warning>
