The **PF2e for Foundry VTT** system brings Pathfinder Second Edition to life inside [Foundry Virtual Tabletop](https://foundryvtt.com). It automates the rules that define PF2e gameplay: action economies, condition tracking, spellcasting, degree-of-success calculation, compendium-backed character building, and much more.

The system is built and maintained by a volunteer development team under a partnership agreement between Foundry Gaming LLC and Paizo Inc. It is published free of charge with the endorsement of Foundry Gaming LLC.

## What the system provides

Once installed, the PF2e system gives you a complete digital ruleset:

- **Actor types** for player characters, NPCs, familiars, hazards, vehicles, armies, loot containers, and parties
- **Item types** covering spells, feats, ancestries, heritages, classes, equipment, conditions, deities, effects, and more
- **Rule Elements** — a flexible rules engine that lets items modify actor statistics and automate mechanical effects without requiring custom code
- **Compendium library** with hundreds of packs drawn from Paizo rulebooks, adventure paths, standalone adventures, and Pathfinder Society content
- **Automated rolls** for checks, saves, and attacks — including full modifier stacking, roll notes, and degree-of-success outcomes
- **Homebrew support** for custom traits, damage types, languages, and system settings

## Who maintains the system

The PF2e system is an open-source volunteer project. The development team accepts contributions from the community through GitHub pull requests, and new contributors are encouraged to join the conversation in the **#pf2e channel** on the [Foundry Discord server](https://discord.gg/foundryvtt).

The repository is hosted at [github.com/foundryvtt/pf2e](https://github.com/foundryvtt/pf2e). Bug reports and feature requests are tracked through [GitHub Issues](https://github.com/foundryvtt/pf2e/issues).

## Paizo partnership

All Pathfinder Second Edition rules content included in the system is used with permission through the formal partnership between Foundry Gaming LLC and Paizo Inc. Content that falls under Paizo's Open Game Content is additionally covered by the Open Game License.

The system does not require you to own physical Paizo products to use with Foundry VTT, although owning the rulebooks enables you to reference the full rules text outside the virtual tabletop.

## Licensing

The project uses a split licensing model:

| Component | License |
|---|---|
| All HTML, CSS, and JavaScript/TypeScript source code | Apache License v2.0 |
| Game system rules and mechanics (OGL content) | Open Game License v1.0a |
| Pathfinder content used with Paizo's permission | Foundry Gaming LLC / Paizo Inc. partnership agreement |
| Foundry VTT system module | [Limited License Agreement for module development](https://foundryvtt.com/article/license/) |

Art license information for assets included in the system is provided in the `./packs/` folder alongside the compendium data that references it.
